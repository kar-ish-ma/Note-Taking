# NoteFlow
## Description
### Note-Flow is a simple and efficient web application designed to help users jot down, organize, and manage their notes. Built with modern web technologies, it provides a clean and intuitive interface for creating, editing, and deleting notes, making it an ideal tool for personal or professional use.

## Features
### Create Notes: Add new notes with a title and description.
### Delete Notes: Remove unwanted notes.

## How to Use
### Add a Note: Use the "Add Note" button to create a new note by entering a title and content.
### Delete a Note: Use the "Delete" button to remove a note.

## Technologies Used
### React: Component-based UI development.
### CSS: Styling the interface 
### JavaScript: Core functionality and logic.
